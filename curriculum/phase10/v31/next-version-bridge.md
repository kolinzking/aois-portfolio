# v31 Next Version Bridge

Authoring status: authored

## What This Version Unlocks

v31 gives AOIS-P a multimodal frontier contract:

- image, audio, video, and document modality catalog
- media intake and validation controls
- consent, privacy, redaction, accessibility, and transcript gates
- cross-modal consistency checks
- evidence and confidence handling
- text fallback behavior
- platform and release integration

## Why The Next Version Exists

v32 adds edge and offline inference patterns.

The bridge from v31 to v32 is:

```text
If AOIS-P can reason over non-text signals, the next constraint is where that
reasoning runs when connectivity, compute, memory, latency, and data locality
are limited.
```

v32 will compare centralized and edge paths and explain which constraints
dominate edge deployments.
