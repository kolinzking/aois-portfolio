# v28 Looking Forward

Authoring status: authored

## What You Should Carry Forward

Carry forward these v28 controls:

- source and branch review
- least-privilege workflow permissions
- dependency, test, policy, and security gates
- artifact digest
- build provenance
- artifact signing and signature verification
- environment approval
- rollout health
- rollback readiness
- staged model rollout
- guarded feature flags

## What The Next Version Will Build On

v29 adds experiment and model delivery tracking.

v28 can decide whether a release candidate should promote, block, or hold. v29
adds evidence over time: which model or behavior changed, what experiment
compared it, what rollout evidence was collected, and why the change was
accepted or rejected.
