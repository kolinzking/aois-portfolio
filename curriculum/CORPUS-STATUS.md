# AOIS Curriculum Corpus Status

This file tracks the authoring state of the full lesson corpus.

Status labels:

- `authored` = lesson pack exists with real version-specific content
- `partial` = some real content exists, but the lesson pack is incomplete
- `scaffolded` = placeholder lesson pack exists, but the version has not yet been fully authored

## Current Rule

Live teaching begins only after the corpus is fully authored and reviewed end to end.
