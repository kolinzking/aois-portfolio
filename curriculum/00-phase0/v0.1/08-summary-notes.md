# v0.1 Summary Notes

Authoring status: authored

## What Was Built

The first AOIS operating layer: direct Linux inspection habits.

## What Was Learned

Linux inspection is the first operational layer of AOIS.
The shell is not just a place to run commands; it is how you see the machine.

## Core Limitation Or Tradeoff

Linux gives visibility, not interpretation.
That is exactly why Bash scripting comes next.
