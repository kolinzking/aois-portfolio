# v14.5 Looking Forward

Authoring status: authored

## What You Should Carry Forward

Carry forward these rules:

1. Cache key design is security design.
2. TTL and invalidation are mandatory.
3. Tenant isolation is mandatory.
4. Hit rate does not prove correctness.
5. Live cache infrastructure needs approval.

## What The Next Version Will Build On

`v15` builds on performance engineering by introducing fine-tuning and adaptation decisions.
