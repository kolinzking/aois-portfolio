## Start Here

Start with [introduction.md](introduction.md), then work through [notes.md](notes.md).

This version teaches GPU infrastructure operations without applying resources.

The goal is to understand the infrastructure layer that must exist before GPU inference can run safely.

## Self-Paced Path

1. Read the introduction.
2. Inspect `inference/aois-p/gpu-infrastructure-operations.plan.json`.
3. Run the validator.
4. Complete the ops and break labs.
5. Answer the mastery checkpoint in [notes.md](notes.md#mastery-checkpoint).

Do not continue if you cannot explain operator, device plugin, scheduling, MIG awareness, and GPU observability.
