# v14 Looking Forward

Authoring status: authored

## What You Should Carry Forward

Carry forward these rules:

1. Throughput must be judged with latency.
2. Queueing must have backpressure.
3. Concurrency must have limits.
4. Cache behavior must have correctness rules.
5. Simulation is not production benchmarking.

## What The Next Version Will Build On

`v14.5` builds on this by focusing on performance engineering and caching.
