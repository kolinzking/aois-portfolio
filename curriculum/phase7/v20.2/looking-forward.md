# v20.2 Looking Forward

Authoring status: authored

## What You Should Carry Forward

AOIS can now plan a tool-using response, account for its cost, and choose a
budget-aware route before spend happens.

Carry forward these controls:

- route before spend
- preserve budget reserve
- skip tools when confidence and evidence are enough
- pause expensive non-high-severity branches
- block routing when accounting is incomplete

## What The Next Version Will Build On

v21 builds on this by governing the tool catalog itself. Once routes can select
tools, AOIS needs registry ownership, schemas, scopes, lifecycle, and audit rules
for those tools.
