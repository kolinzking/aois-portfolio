# v25 Next Version Bridge

Authoring status: authored

## What This Version Unlocks

v25 completes Phase 7.

AOIS-P now has:

- governed tool planning
- cost accounting
- budget-aware routing
- governed MCP tool exposure
- durable workflow state
- stateful orchestration
- connected agent evaluation
- runtime autonomy control
- multi-agent collaboration policy
- safe execution boundaries

## Why The Next Version Exists

Phase 8 turns AOIS into a product surface.

The system now has enough governed operational state to make a useful human
interface: incidents, traces, decisions, approvals, budgets, agent roles,
autonomy mode, and execution-boundary outcomes.

The bridge from v25 to v26 is:

```text
bound execution -> show operators what AOIS is doing and why
```

v26 begins Phase 8 with dashboard and real-time visibility.
