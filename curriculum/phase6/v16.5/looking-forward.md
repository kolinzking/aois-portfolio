# v16.5 Looking Forward

Authoring status: authored

## What You Should Carry Forward

Carry forward these rules:

1. Multi-step behavior needs step traces.
2. Parent-child relationships preserve causality.
3. Decision reasons are operational data.
4. Tool calls need tool call IDs.
5. Trace summaries must be redacted.

## What The Next Version Will Build On

`v17` builds on incident traceability by introducing event streaming.
