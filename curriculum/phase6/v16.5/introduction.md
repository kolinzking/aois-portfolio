# v16.5 Introduction

Authoring status: authored

## What This Version Is About

`v16.5` is about tracing multi-step incident and agent-style behavior.

It adds step-level trace planning on top of the unified telemetry foundation from `v16`.

## Why It Matters In AOIS

Final responses do not explain how AOIS reached a decision.

Incident and future agent workflows need step timelines, decision reasons, timings, parent-child relationships, and redacted summaries.

## How To Use This Version

Run the validator and simulator locally.

Focus on following the incident timeline from ingest to response.

Do not start an agent runtime, execute tools, call providers, or run telemetry backends.
