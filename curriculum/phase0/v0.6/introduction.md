# v0.6 Introduction

`v0.6` is the first time AOIS becomes a service instead of only a collection of scripts and modules.

## What You Will Build

You will work through:

- [app/main.py](../../../app/main.py)
- [app/analysis.py](../../../app/analysis.py)

and expose:

- `GET /health`
- `POST /analyze`

## Why It Matters In AOIS

Every later AOIS phase assumes a service boundary exists.
This version gives you that boundary before AI arrives.

## How To Use This Version

Start with [notes.md](notes.md), run [lab.md](lab.md), keep [runbook.md](runbook.md) as the quick reference, then close with [summarynotes.md](summarynotes.md) and [looking-forward.md](looking-forward.md).
