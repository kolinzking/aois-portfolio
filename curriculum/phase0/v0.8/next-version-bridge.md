# v0.8 Next Version Bridge

Authoring status: authored

## What This Version Unlocks

`v0.8` completes the Phase 0 foundation by adding persistence design.

AOIS now has:

- machine inspection habits
- shell automation
- Git discipline
- HTTP inspection
- deterministic Python analysis
- local API boundary
- LLM request planning
- database schema design

## Why The Next Version Exists

The next step is the Phase 0 capstone.

The capstone should prove that the whole foundation can be explained and practiced as one system, not as disconnected lessons.

After the capstone, Phase 1 can begin moving from foundations toward deployable service infrastructure.
