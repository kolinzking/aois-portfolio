# v0.3 Looking Forward

Authoring status: authored

## What You Should Carry Forward

Every AOIS change should be inspectable.

Before you commit, know:

- what changed
- why it changed
- whether the files belong together

## What The Next Version Will Build On

`v0.4` builds on Git discipline by adding HTTP and networking inspection.
Once AOIS exposes services, you need to inspect behavior across boundaries, not only inside files.
